## 1. 相关内容
- `ahCheck.py` #主模块代码

这一行配置要签到的账号信息，发件邮箱和收件邮箱自行更改

```json
{
	"userIn.userPasswd": "azPGGyxxxxx",  				#加密后密码
	"userIn.phone_no": "152xxx7101",  					#手机号
	"token": "7215751b1ed2de841fd705456xxxx",  			#token
	"mail": ""  										#收件邮箱
}
```

- 每日自动签到和摇一摇

- APP签到一定天数及摇一摇可以获得流量红包
